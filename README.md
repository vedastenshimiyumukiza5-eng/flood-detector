# Flood Detector — Smart Flood Detection & Early Warning Solutions

A responsive website for a comprehensive flood detection and early warning system designed to save lives and protect communities.

## Pages

- **Home (index.html)** — Hero section, solution overview, how the system works, meet the team
- **Solution (solution.html)** — Technical details on sensors, monitoring, and alerts
- **USSD Alerts (ussd.html)** — Information about USSD-based alert system
- **Impact (impact.html)** — Real-world use cases and impact metrics
- **About (about.html)** — Company mission, vision, and team information
- **Contact (contact.html)** — Partnership and inquiry form

## Getting Started

1. Place all files in the same directory and create an `images/` folder for media assets
2. Open `index.html` in your web browser to view the site
3. Replace placeholder images with actual photos and illustrations

## Deployment

The site can be deployed to any static hosting service:
- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting

For the contact form, integrate with a backend service or form handler (Formspree, Netlify Forms, etc.)
