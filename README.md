# Flood Detector — Static Website

A responsive static website (HTML, CSS, JavaScript) for the Flood Detector early-warning solution.

## Pages

- **Home** (`index.html`) — hero, solution cards, how-it-works timeline, meet the team
- **Solution** (`solution.html`) — technology details
- **USSD Alerts** (`ussd.html`) — USSD/SMS feature demo
- **Impact** (`impact.html`) — use cases and stats
- **About** (`about.html`) — company info
- **Contact** (`contact.html`) — contact form

## Setup

1. Place all files in the same directory. Add an `images/` folder for photos.
2. Open `index.html` in a browser to preview.
3. Replace placeholder images (`images/*.jpg`) with real photos or illustrations.

## Deployment

Hosted on Netlify as a static site. The contact form can be connected to Netlify Forms or a backend service.
